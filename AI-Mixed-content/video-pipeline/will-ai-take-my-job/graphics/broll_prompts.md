# B-Roll & Image Generation Prompts

Base style block (append to every prompt):
```
ultra-clean futuristic AI interface, holographic elements, glowing data streams,
minimalistic tech environment, soft blue and white lighting, cinematic depth,
8k detail, no humans, no text
```

Negative prompt (use for all):
```
humans, faces, text, watermark, logo, political symbols, dystopian imagery,
violence, noise, blur, low quality, cartoon
```
